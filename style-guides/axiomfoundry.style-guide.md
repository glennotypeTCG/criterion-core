# Axiom Foundry Style Guide

## Status

Placeholder

## Purpose

This Style Guide will govern formation-related materials for Axiom Foundry.

## Applies To

- Axiom Foundry Sales Deck
- Axiom Foundry Pitch Deck
- Formation offers
- Founder conversations
- Operating cadence materials

## Current Rule

Axiom Foundry should feel:

- Disciplined
- Operator-led
- Protective
- Structured
- Calm
- High-agency
- Venture-aware

It should not feel:

- Controlling
- Predatory
- Like outsourced labor
- Like mandatory consulting
- Like bureaucracy
- Like a traditional agency or consultancy
