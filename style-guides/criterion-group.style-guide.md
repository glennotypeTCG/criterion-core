# Criterion Group Style Guide

## Status

Placeholder

## Purpose

This Style Guide will govern the voice, posture, visual direction, and messaging rules for Criterion Group system-facing materials.

## Applies To

- Criterion Group Idea Deck
- Internal system explanations
- Strategic architecture conversations
- Coordination-layer narratives

## Current Rule

Criterion Group should feel:

- Strategic
- Clear
- Coordinated
- Human
- Structurally advantaged
- Legally disciplined

It should not feel:

- Like a public ecosystem
- Like a holding company unless legally true
- Like a guaranteed funnel
- Like a vague innovation consultancy
