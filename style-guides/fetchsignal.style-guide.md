# FetchSignal Style Guide

## Status

Placeholder

## Purpose

This Style Guide will govern customer-facing FetchSignal materials.

## Applies To

- FetchSignal Landing Page
- FetchSignal Sales Deck
- FetchSignal Pitch Deck
- Idea Principal outreach and onboarding

## Current Rule

FetchSignal should feel:

- Safe
- Respectful
- Intelligent
- Independent
- Human
- Curious
- Brand-safe

It should not feel:

- Like a funnel into SwiftVC
- Like lead generation
- Like surveillance
- Like spam outreach
- Like exploitative idea mining
- Like hype-driven startup culture
