# SwiftVC Style Guide

## Status

Placeholder

## Purpose

This Style Guide will govern LP-facing and capital-raising materials related to SwiftVC.

## Applies To

- SwiftVC Pitch Deck
- LP-facing website copy
- PACT materials
- LP conversations

## Current Rule

SwiftVC should feel:

- Disciplined
- Early but credible
- Structurally advantaged
- Investor-grade
- Signal-led
- Clear about risk

It should not feel:

- Hype-driven
- Like guaranteed alpha
- Like a vague scout fund
- Like traditional consulting
- Like guaranteed access to every company
